# 캐스팅엔 ZENO 맞춤 영업 제안서 자동 생성기

## 역할
너는 캐스팅엔 영업팀을 위한 ZENO 맞춤 제안서 생성 Streamlit 앱을 만드는 풀스택 개발자다.
영업팀은 매 고객사 미팅 전에 "이 회사에 ZENO의 어떤 기능이 맞냐, ROI 얼마냐"를
커스텀 제안서로 만들어야 한다. 고객사 정보를 입력하면 맞춤 제안서 Word를 자동 생성한다.

## 앱 개요
- 단일 페이지, 왼쪽 입력 폼 + 오른쪽 미리보기 구조
- 파일명: app.py
- 실행 위치: Castingn-AI/12.ZENO제안서생성기/code/

## 기술 스택
- streamlit
- openai (GPT-4o-mini)
- python-docx
- python-dotenv
- plotly (ROI 시각화)

# UI 레이아웃

## 헤더
- 캐스팅엔 로고 대신 텍스트 로고 + 태그라인 ("AI-Native Spend Management")
- 페이지 제목: "ZENO 맞춤 제안서 생성기"

## 메인 레이아웃: col1 (입력, 40%) | col2 (미리보기, 60%)

### col1: 입력 폼 (st.form)

#### 섹션 1: 고객사 기본 정보
- 고객사명 (text_input, 필수)
- 업종 (selectbox): 제조/물류/유통/IT/금융/건설/F&B프랜차이즈/서비스/기타
- 직원수 (number_input)
- 전국 사업장/지점 수 (number_input)
- 연간 간접구매 추정액 (number_input, 단위: 억원)

#### 섹션 2: 현재 구매 방식
- 현재 구매 방식 (radio): 완전분산/부분중앙화/중앙화됨
- 현재 사용 시스템 (multiselect): ERP없음/SAP/Oracle/더존/자체개발/기타
- 평균 구매 승인 소요일 (number_input, 단위: 일)

#### 섹션 3: 주요 페인포인트 (multiselect, 최대 3개 선택)
- 분산 발주로 비용 통제 안 됨
- 승인 프로세스가 너무 오래 걸림
- 공급업체 관리가 체계적이지 않음
- 구매 데이터/리포트가 없음
- 부서마다 따로 구매해서 협상력 없음
- 감사/컴플라이언스 대응 어려움
- ERP와 구매 시스템 연동 안 됨

#### 섹션 4: 미팅 정보
- 미팅 날짜 (date_input)
- 담당 영업 (text_input)
- 미팅 상대방 직책 (selectbox): CPO/CFO/구매팀장/IT팀장/경영기획팀장/대표

- [제안서 생성] 버튼

### col2: 미리보기
- 생성 전: 입력 안내 placeholder
- 생성 후: 제안서 섹션별 미리보기 (expander)

# GPT 처리 로직

## 단계 1: ZENO 기능 매핑
```python
def map_features(고객사_정보, 페인포인트):
    """
    페인포인트 기반으로 ZENO_FEATURES에서 가장 적합한 3~5개 기능 선택
    """
    # 페인포인트 → 적합 기능 매핑 로직
    relevant_features = []
    for feature, info in ZENO_FEATURES.items():
        match_score = sum(1 for p in 페인포인트 if p in str(info['적합']))
        if match_score > 0:
            relevant_features.append((feature, info, match_score))
    return sorted(relevant_features, key=lambda x: x[2], reverse=True)[:5]
```

## 단계 2: ROI 자동 계산
```python
def calculate_roi(annual_spend_억, employees, 현재_승인소요일, 사업장수):
    """
    캐스팅엔 실제 사례 기반 ROI 계산
    - 평균 12.4% 비용 절감
    - 승인 처리 3배 단축
    - 6개월 ROI 회수
    """
    연간_절감액 = annual_spend_억 * 100000000 * 0.124
    월간_절감액 = 연간_절감액 / 12
    승인_단축일 = 현재_승인소요일 * (2/3)  # 3배 단축

    return {
        "연간_절감액": 연간_절감액,
        "월간_절감액": 월간_절감액,
        "승인_단축일수": 승인_단축일,
        "ROI_회수_개월": 6,  # 캐스팅엔 평균
    }
```

   - ROI 회수: 6개월

   - 도입 전/후 연간 지출 비교 바차트

   - 업종이 비슷한 도입 사례 1~2개

5. **제안서 미리보기** (섹션별 expander)

6. **[Word 제안서 다운로드]**
   - 파일명: f"{고객사명}_ZENO제안서_{오늘날짜}.docx"
   - 커버페이지 포함 (고객사명, 날짜, 담당 영업)

7. **[미팅 준비 체크리스트 출력]**
   - "미팅 상대방이 {직책}이니까 관심 포인트: ..."
   - 예상 질문 5개 + 답변 가이드

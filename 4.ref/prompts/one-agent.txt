4주차. one-agent.py
4개 통합 앱 만들기(one-agent.py)

# 목적
- one.py 프로그램을 ai agent를 활용해서 4개의 tools을 만들어 통합하는 것입니다.

# 생성 파일명
- one-agent.py

# 화면UI 및 요구사항
- 왼쪽에 side bar를 만들어서 사용자의 질문에 따라서 AI Agent가 다음 1개를 선택하도록 합니다. 
- 기존 one.py에 있었던 수동 선택을 모두 제거합니다. 
- 사용자가 질문을 하면, 질문을 처리하기 위해서 선택한 tool 이름을 기억합니다.
- 이를 위해서 callback handler를 사용합니다. 
- 선택된 tool 이름을 sidebar 에 표시합니다.
- tool 이름: 시간, 챗봇, 인터넷검색, RAG 
- 시간을 선택하면 아래 time.py를 실행해주세요.
- 챗봇을 선택하면 아래 chatbot.py를 실행해주세요
- 인터넷 검색을 선택하면 아래 internet.py를 실행해주세요. 
- RAG를 선택하면 아래 rag.py를 실행해주세요.
- 해당 python 코드들은 3.first/code에 있습니다.

# 간단한 RAG 챗봇 만들기

## 목적
- Streamlit을 활용하여 간단한 rag 챗봇을 만들기입니다.

## 생성 파일명
- rag.py

## 요구사항
Langchain 을 활용해서 RAG 어플리케이션을 streamlit으로 구현하는 코드를 생성해주세요

## 주의사항
- LLM 사용을 위한 API KEY는 .env에 있으니 반드시 python-dotenv의 load_dotenv()를 사용해 로드하세요.
- .env 경로는 Castingn-AI/.env 입니다. 실행 위치와 무관하게 절대경로 기준으로 명시적으로 로드하세요.
- 환경변수가 없으면 예외로 종료하지 말고 사용자에게 키 누락 메시지를 보여주세요.
- Streamlit을 사용하여 대화형 웹인터페이스를 제공합니다.
- 이전 대화는 기억해야 합니다. 
- vector db는 FAISS를 사용합니다
- 여러개의 pdf 파일을 받고 처리할 수 있도록 합니다


## 시스템 메시지
- 답변을 생성할 때, 다음과 같은 시스템 메시지를 사용하세요 
“너는 매우 친절한 선생님이야. 답변은 매우 쉽게 중학생 레벨에서 이해할 수 있도록 해줘. 
그러나 내용은 생략하는 것 없이 모두 답을 해줘. 모르면 모른다고 답해줘. 말투는 존대말 한글로 해줘.”

## 상세절차
- 왼쪽 화면(side bar)에서 pdf 파일 명을 받습니다. 
- PDF 파일에서 텍스트를 추출합니다
- 화면에 다른 안내는 필요없습니다.
- 추출된 텍스트를 작은 청크로 나눕니다.(chunk size = 1000, overlap = 200)
- 임베딩은 openai embedding을 사용합니다.
- 텍스트 청크를 벡터 데이터베이스에 저장합니다. 
- 저장이 끝나면 저장이 끝났다는 메시지를 화면에 보냅니다.
- 사용자의 질문을 받습니다. (사용자의 질문은 화면 오른쪽 맨아래 고정입니다)
- 사용자의 질문에 대해 관련 정보를 벡터 데이터베이스에서 검색합니다.
- 검색은 ensemble retriever 를 사용합니다.
- 검색된 정보를 바탕으로 주의사항에서 정의된 LLM을 사용하여 답변을 생성합니다.

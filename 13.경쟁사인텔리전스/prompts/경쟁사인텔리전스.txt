# 캐스팅엔 경쟁사 인텔리전스 대시보드

## 역할
너는 캐스팅엔 기획·마케팅팀을 위한 경쟁사 인텔리전스 Streamlit 앱을 만드는 풀스택 개발자다.
BSM(Business Spend Management) 시장의 경쟁사 동향, 투자 뉴스, 신규 기능 출시 정보를
자동으로 수집·분석하여 주간 인텔리전스 리포트를 생성한다.
2027 KOSDAQ IPO를 앞둔 캐스팅엔의 시장 포지셔닝 자료로 활용한다.

## 앱 개요
- 단일 페이지, 사이드바 설정 + 메인 대시보드 구조
- 파일명: app.py
- 실행 위치: CastingN/13.경쟁사인텔리전스/code/

## 기술 스택
- streamlit
- openai (GPT-4o-mini, web_search tool 사용)
- plotly
- python-docx
- python-dotenv
- requests (검색 API 직접 호출)

# 수집 및 분석 로직

## GPT web_search 활용
```python
def collect_intelligence(competitors, keywords, period):
    """
    GPT-4o의 web_search 도구를 활용해 경쟁사 정보 수집
    """
    client = openai.OpenAI()

    collected = []

    for competitor, info in competitors.items():
        for keyword in info['검색키워드']:
            response = client.chat.completions.create(
                model="gpt-4o-mini",
                tools=[{"type": "web_search_preview"}],
                messages=[{
                    "role": "user",
                    "content": f"""
                    최근 {period} 동안 다음 키워드로 검색하여 중요한 뉴스나 정보를 수집해주세요:
                    키워드: {keyword}

                    다음 JSON으로 응답:
                    {{
                      "검색키워드": "{keyword}",
                      "경쟁사": "{competitor}",
                      "수집결과": [
                        {{
                          "제목": "",
                          "날짜": "",
                          "출처": "",
                          "요약": "",
                          "중요도": "HIGH/MEDIUM/LOW",
                          "카테고리": "신제품/투자/고객사/가격/채용/기타"
                        }}
                      ]
                    }}
                    결과가 없으면 수집결과를 빈 배열로 응답.
                    """
                }]
            )
            # 응답 파싱 및 collected에 추가

    return collected
```

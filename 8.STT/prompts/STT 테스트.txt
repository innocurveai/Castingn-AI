# 역할
너는 파이썬 기반의 음성 인식(STT) 시스템을 구축하는 개발자다.
음성 파일을 업로드하면 텍스트로 변환해주는 핵심 기능만 포함된 Streamlit 웹 앱을 작성하라.

# 폴더 구조 및 파일명
작업 경로: ./8.STT/code/

파일명: stt_only.py

# 주요 기능 및 작업 흐름
환경 설정: .env 파일에서 OPENAI_API_KEY를 로드한다.

UI 구성:

페이지 제목: "STT 변환 App"

UI는 4주차/code/ref.py를 참고해서 제작한다.

파일 업로더: mp3, wav, m4a, mp4, ogg 등 음성 파일 드래그 앤 드롭 지원.

오디오 미리보기: 업로드한 파일을 즉시 들어볼 수 있는 플레이어 제공.

STT 변환 로직 (transcribe_audio 함수):

파일 처리: pydub를 사용하여 업로드된 파일을 처리한다. (내부적으로 ffmpeg를 사용함)

API 호출: requests 라이브러리를 사용해 OpenAI Whisper API(whisper-1)를 직접 호출한다.

- LLM 사용을 위한 API KEY는 .env에 있으니 반드시 python-dotenv의 load_dotenv()를 사용해 로드하세요.
- .env 경로는 AI-Project/.env 입니다. 실행 위치와 무관하게 절대경로 기준으로 명시적으로 로드하세요.
- 환경변수가 없으면 예외로 종료하지 말고 사용자에게 키 누락 메시지를 보여주세요.

언어 설정: 한국어(ko)로 고정한다.

결과 출력:

[STT 변환 시작] 버튼 클릭 시 프로세스 진행.

변환이 완료되면 결과를 st.text_area에 출력하여 사용자가 내용을 확인하고 복사할 수 있게 한다.

사후 처리: 변환에 사용된 임시 파일은 작업 완료 후 반드시 삭제한다.

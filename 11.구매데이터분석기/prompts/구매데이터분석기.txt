# 캐스팅엔 구매 데이터 분석기

## 역할
너는 캐스팅엔 고객성공팀·거버넌스팀을 위한 구매 데이터 분석 Streamlit 앱을 만드는 풀스택 개발자다.
탭 1에서는 고객사 지출 데이터를 분석해 절감 기회를 찾고 (캐스팅엔의 핵심 컨설팅 상품),
탭 2에서는 동일 데이터에서 이상거래·낭비 패턴을 자동 탐지한다 (ZENO DGF 거버넌스 핵심 가치).

## 앱 개요
- 탭 1: 간접구매 현황 진단기
- 탭 2: 구매 이상거래 탐지기
- 파일명: app.py
- 실행 위치: Castingn-AI/11.구매데이터분석기/code/

## 기술 스택
- streamlit
- pandas
- plotly
- openai (GPT-4o-mini)
- reportlab (PDF 리포트)
- python-dotenv

## 프로젝트 구조
```
Castingn-AI/11.구매데이터분석기/
├── code/
│   └── app.py

```

## 입력 데이터 형식 (CSV)
```
날짜,부서,카테고리,공급업체,금액,결재자,비고
2025-01-05,마케팅팀,사무용품,A문구,450000,김팀장,
2025-01-05,마케팅팀,사무용품,B문구,480000,이대리,  ← 같은 날 같은 카테고리 분산발주
2025-01-15,IT팀,IT장비,C전자,4950000,박팀장,  ← 결재한도 500만원 직전 금액 (쪼개기 의심)
```

# 탭 1: 간접구매 현황 진단기

## 배경
ZENO 신규 도입 예정 고객사가 "도입 전 현황 분석해달라"고 3개월치 지출 내역 엑셀을 보낸다.
컨설팅팀이 낭비 항목, 통합 기회, 예상 절감액을 분석해서 진단 리포트를 만든다.
이게 캐스팅엔이 고객사에게 파는 컨설팅 서비스의 핵심이다.

## 입력 UI
- CSV 파일 업로드 또는 샘플 데이터 사용 버튼
- 고객사명 (text_input)
- 분석 기간 표시 (업로드 후 자동 감지)
- 회사 규모 (columns 2개)
  - 직원수 (number_input)
  - 사업장 수 (number_input)
- [진단 시작] 버튼

## 분석 로직

### 기본 통계
```python

# 카테고리별 지출

# 부서별 지출

# 공급업체 집중도

top5_ratio = by_vendor.nlargest(5).sum() / total_spend  # 상위 5개 업체 집중도
```

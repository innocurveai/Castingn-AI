# 캐스팅엔 공급업체 평가 플랫폼

## 역할
너는 캐스팅엔 소싱팀을 위한 공급업체 평가 Streamlit 앱을 만드는 풀스택 개발자다.
탭 1에서는 여러 업체가 서로 다른 양식으로 보내온 견적서 Excel을 정규화·비교하고,
탭 2에서는 WorkMarket9 등록을 신청한 신규 업체를 자동으로 심사·스코어링한다.

## 앱 개요
- 탭 1: 견적 자동 비교 분석기
- 탭 2: 신규 공급업체 심사 스코어링
- 파일명: app.py
- 실행 위치: Castingn-AI/10.공급업체평가플랫폼/code/

## 기술 스택
- streamlit
- pandas
- openpyxl
- plotly
- openai (GPT-4o-mini)
- python-docx
- python-dotenv

## 프로젝트 구조
```

├── code/
│   └── app.py

    └── 견적서_샘플_C업체.csv
```

## 환경 변수
- .env 위치: Castingn-AI/.env
- OPENAI_API_KEY 필수

# 탭 1: 견적 자동 비교 분석기

## 배경
A업체는 품목코드 기준, B업체는 제품명 기준, C업체는 모델번호 기준으로 각각 다른 양식의 견적서를 보낸다.
소싱팀이 이를 손으로 비교표 만들고 낙찰 추천 보고서 작성하면 1시간 30분 소요.
AI가 양식을 정규화하고 비교·추천까지 처리한다.

## 입력 UI
- 견적서 Excel 다중 업로드 (file_uploader, accept_multiple_files=True)
  - 지원 형식: xlsx, xls, csv
  - 최대 10개
- 구매 품목명 (text_input, 예: "사무용 의자")
- 평가 가중치 (st.columns으로 슬라이더 4개 나란히)
  - 가격: 기본 40%
  - 납기: 기본 30%
  - 브랜드/제조사: 기본 15%
  - AS/보증: 기본 15%
  - 합계 100% 자동 검증 (합계 != 100이면 경고)
- [비교 분석 시작] 버튼

## 처리 로직

### Step 1: 각 파일 GPT로 정규화
각 업로드 파일의 내용을 GPT에게 보내 표준 스키마로 변환:
```python
표준_스키마 = {
    "업체명": str,
    "품목명": str,
    "모델명": str,
    "단가": int,      # 원 단위
    "수량": int,
    "총액": int,
    "납기": str,      # 예: "2주"
    "보증기간": str,   # 예: "1년"
    "AS조건": str,
    "비고": str
}
```

GPT 프롬프트:
```
다음 Excel 데이터를 분석하여 표준 견적 스키마로 변환해주세요.
업체명: {파일명에서 추출}
데이터: {DataFrame.to_string()}

JSON 배열로 응답하세요. 각 행이 하나의 품목입니다.
알 수 없는 값은 "" 또는 0으로 처리하세요.
```

1. 정규화 완료 알림 (st.success)

4. 추천 업체 카드 (st.metric으로 강조)
5. 리스크 경고 (st.warning)
6. [구매 결재 보고서 Word 다운로드] 버튼
   - 파일명: f"견적비교보고서_{품목명}_{오늘날짜}.docx"

# 탭 2: 신규 공급업체 심사 스코어링

## 배경
WorkMarket9에는 7,800개 검증 파트너, 전체 평가 업체 29,000개가 있다.
새 공급업체 등록 신청이 들어오면 소싱팀이 수기로 심사한다.
표준화된 평가 기준으로 AI가 자동 점수화하고 등록 권고 여부를 결정한다.

## 평가 기준 (캐스팅엔 WorkMarket9 기준 반영)
| 항목 | 배점 |
|------|------|
| 사업 연수 | 15점 |
| 주요 납품 실적 (기업수/규모) | 20점 |
| 취급 카테고리 전문성 | 15점 |
| 가격 경쟁력 (샘플 견적 기준) | 20점 |
| 인증/자격증 보유 | 10점 |
| 배송/납기 능력 | 10점 |
| 재무 건전성 (매출/직원수) | 10점 |

## 입력 UI
st.form으로 구성:
- 업체명 (text_input)
- 대표자명 (text_input)
- 사업자등록번호 (text_input)
- 설립연도 (number_input)
- 직원수 (number_input)
- 연매출 (number_input, 단위: 억원)
- 취급 카테고리 (multiselect): 사무용품/IT장비/청소용역/물류/시설관리/소모품/인쇄/가구/전기용품/기타
- 주요 납품 기업 (text_area, 최대 5개 기입)
- 인증 보유 여부 (multiselect): ISO9001/ISO14001/기업부설연구소/벤처확인/없음
- 납기 리드타임 (selectbox): 당일/1~2일/3~5일/1주일 이상
- 특이사항/강점 (text_area)
- [심사 시작] 버튼

## GPT 처리 로직
```
당신은 캐스팅엔 WorkMarket9의 공급업체 심사 전문가입니다.
아래 업체 정보를 분석하여 등록 심사 점수를 산정하고 의견을 작성하세요.

업체 정보: {입력 데이터}

평가 기준:
- 사업 연수 (15점): 10년 이상=15점, 5~10년=10점, 3~5년=7점, 1~3년=4점, 1년 미만=1점
- 납품 실적 (20점): 대기업 납품=+5점, 실적 다양성, 최근 3년 기준
- 카테고리 전문성 (15점): 전문 카테고리 수, 집중도
- 가격 경쟁력 (20점): 제공된 정보 기반 추정
- 인증 보유 (10점): 인증 1개당 3점, 최대 10점
- 납기 능력 (10점): 당일=10점, 1~2일=8점, 3~5일=5점, 1주일+=2점
- 재무 건전성 (10점): 매출/직원수 기반 추정

JSON으로 응답:
{{
  "항목별_점수": {{
    "사업연수": 0,
    "납품실적": 0,
    "카테고리전문성": 0,
    "가격경쟁력": 0,
    "인증보유": 0,
    "납기능력": 0,
    "재무건전성": 0
  }},
  "총점": 0,
  "등급": "A/B/C/D",
  "권고": "즉시등록/조건부등록/보류/거절",
  "강점": [],
  "우려사항": [],
  "심사의견": ""
}}
```

## 등급 기준
- A (85점 이상): 즉시 등록, 우선 파트너 후보
- B (70~84점): 즉시 등록
- C (55~69점): 조건부 등록 (추가 서류 요청)
- D (54점 이하): 보류 또는 거절

## 출력 UI
1. 총점 게이지 (st.progress)
2. 등급 배지 (색상 강조)
3. 항목별 점수 레이더 차트 (plotly)
4. 강점 / 우려사항 두 컬럼 병렬 표시
5. 심사 의견 서술
6. 권고 결정 (🟢 즉시등록 / 🟡 조건부 / 🔴 보류)
7. [심사 결과 저장] 버튼 (CSV 누적 저장)
   - data/ 폴더에 심사이력.csv 누적
